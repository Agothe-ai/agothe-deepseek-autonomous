# git_summary
Generate a summary of recent Git activity: last N commits, changed files, branch status.

## When to use
When Paul asks "what did I do today", "git summary", "recent commits", "what changed".

## Tags
git, commits, history, summary, changelog

## Parameters (optional, via context)
- context['n'] -- number of commits (default 10)

## Output
Returns formatted git log with stats.
