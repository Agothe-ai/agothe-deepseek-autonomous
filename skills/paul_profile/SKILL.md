# paul_profile
Display everything Jarvis knows about Paul: facts, preferences, session stats, skill usage patterns.

## When to use
When Paul asks "what do you know about me", "my profile", "show my data", "paul profile".

## Tags
paul, profile, memory, facts, preferences, identity

## Parameters (via context)
- context['mem'] -- the Jarvis memory dict

## Output
Formatted profile report.
